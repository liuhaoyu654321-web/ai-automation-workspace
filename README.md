# AI 自动化工作台 v1.0

这是一个基于 HTML 前端页面 + n8n + AI 模型搭建的本地 AI 自动化工作台。

本项目把多个常用 AI 自动化工具集中到一个统一入口中，用户可以通过首页进入不同工具，包括长文本总结、自媒体内容生成、学习复盘、客户评论分析、资料总结、PDF 总结和表格数据分析。

## 项目目标

本项目的目标不是做一个完整 SaaS 产品，而是通过多个可运行的小工具，系统练习 AI 自动化项目的完整结构：

- 前端输入
- Webhook 接收
- token 鉴权
- 输入校验
- 数据预处理
- AI 处理
- 空结果兜底
- 统一输出
- 网页展示

## 已集成工具

### 1. AI 长文本总结器

支持输入长文本，自动进行分块总结，并根据不同总结风格输出结果。

适合场景：

- 长文章总结
- 课程笔记总结
- 会议记录整理
- 学习资料压缩

### 2. AI 自媒体内容助手

支持根据原始内容生成小红书笔记、抖音口播稿和标题。

适合场景：

- 小红书图文
- 抖音口播稿
- 内容标题生成
- 学习记录转内容素材

### 3. AI 学习复盘助手

支持输入学习记录，自动整理今日完成事项、遇到的问题、学到的关键知识和下一步计划。

适合场景：

- 每日学习复盘
- 项目复盘
- Notion 学习日志
- 学习打卡文案

### 4. AI 客户评论分析器

支持输入用户评论、客服记录或问卷反馈，自动分析用户痛点、需求、正面反馈、负面问题和改进建议。

适合场景：

- 小红书评论分析
- 抖音评论分析
- 客服记录整理
- 产品反馈分析
- 问卷反馈总结

### 5. AI 资料总结器

支持粘贴文章、课程资料、教程内容或 PDF 复制文本，生成结构化资料总结。

适合场景：

- 文章总结
- 课程笔记整理
- 知识库沉淀
- Notion 学习笔记

### 6. AI PDF 总结器

支持上传可复制文字型 PDF，自动提取文本并生成总结报告。

适合场景：

- PDF 资料总结
- 课程讲义整理
- 电子文档摘要
- 学习材料提炼

当前限制：

- 暂不支持扫描版 PDF
- 暂不支持 OCR
- 长 PDF 还未接入分块总结

### 7. AI 表格数据分析器

支持上传 CSV 表格，自动生成结构化数据分析报告。

适合场景：

- 销售数据分析
- 问卷数据分析
- 客户表分析
- 用户反馈表分析

当前限制：

- 第一版只支持 CSV
- 暂不支持 .xlsx
- 当前只分析前端读取后的 CSV 文本

## 技术结构

项目主要由两部分组成：

### 前端页面

使用 HTML / CSS / JavaScript 编写本地页面，包括：

- 工作台首页
- 各个工具页面
- 表单输入
- 文件上传
- 结果展示
- 复制按钮
- 清空按钮
- 返回工作台按钮

### n8n 工作流

每个工具对应一个 n8n workflow，常见结构为：

```text
Webhook
↓
IF - token 鉴权
↓
IF - 输入校验
↓
数据预处理
↓
AI 节点处理
↓
IF - AI 输出为空检查
↓
Final Output
↓
Respond to Webhook
