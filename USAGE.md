# AI 自动化工作台 v1.0 使用说明

## 访问地址

公网访问地址：

https://liuhaoyu654321-web.github.io/ai-automation-workspace/

## 项目说明

AI 自动化工作台 v1.0 是一个基于 GitHub Pages、n8n Cloud 和 AI 模型搭建的公网演示项目。

用户可以通过工作台首页进入不同工具，提交文本、PDF 或 CSV 数据，由云端 n8n workflow 接收请求、调用 AI 模型处理，并将结果返回到网页展示。

## 已上线工具

当前版本包含 7 个工具：

1. AI 长文本总结器
2. AI 自媒体内容助手
3. AI 学习复盘助手
4. AI 客户评论分析器
5. AI 资料总结器
6. AI PDF 总结器
7. AI 表格数据分析器

## 使用方式

### 1. 打开工作台首页

访问：

https://liuhaoyu654321-web.github.io/ai-automation-workspace/

### 2. 选择工具

在首页点击对应工具卡片中的“进入工具”按钮。

### 3. 输入内容或上传文件

不同工具支持不同输入方式：

- 文本类工具：输入或粘贴文本内容。
- PDF 总结器：上传可复制文字型 PDF。
- 表格数据分析器：上传 CSV 文件。

### 4. 点击提交按钮

根据页面按钮提示，点击“开始总结”“开始分析”“开始复盘”或类似按钮。

### 5. 查看结果

等待 n8n Cloud 调用 AI 模型处理后，结果会显示在网页中。

### 6. 复制结果

每个工具都提供复制按钮，可以将生成结果复制到 Notion、微信、文档或其他地方。

## 各工具说明

### AI 长文本总结器

适合长文章、课程笔记、会议记录、学习资料总结。

### AI 自媒体内容助手

适合根据原始内容生成小红书笔记、抖音口播稿和标题文案。

### AI 学习复盘助手

适合记录每日学习内容、整理学习收获和下一步任务。

该工具已接入 Notion，可将复盘结果保存到 Notion 数据库。

### AI 客户评论分析器

适合分析用户评论、客服记录、问卷反馈中的痛点、需求和改进建议。

### AI 资料总结器

适合整理文章、教程、课程资料和学习笔记。

### AI PDF 总结器

适合上传可复制文字型 PDF，并生成结构化总结。

当前暂不支持扫描版 PDF 和 OCR。

### AI 表格数据分析器

适合上传 CSV 表格，生成数据概览、关键字段、异常问题和业务建议。

当前主要支持 CSV 文件。

## 当前技术结构

当前系统结构为：

GitHub Pages 前端页面
→ n8n Cloud Webhook
→ AI 模型处理
→ Notion / 网页结果返回

其中：

- GitHub Pages：负责托管前端页面。
- n8n Cloud：负责自动化流程处理。
- AI 模型：负责生成总结、分析和内容。
- Notion：用于保存学习复盘历史记录。

## 当前限制

当前版本仍然是 v1.0 演示版，不是完整商业 SaaS。

限制包括：

1. 暂无用户登录系统。
2. 暂无多用户权限管理。
3. 暂无调用次数限制。
4. token 鉴权仍然是基础版本。
5. PDF 总结器暂不支持扫描版 PDF。
6. 表格分析器主要支持 CSV，不支持复杂 Excel。
7. 暂无后台管理系统。
8. 暂无计费系统。

## 维护说明

如果需要修改页面内容：

1. 在本地项目中修改对应 HTML 文件。
2. 使用 Git 提交修改。
3. 推送到 GitHub。
4. GitHub Pages 会自动更新公网页面。

如果需要修改自动化逻辑：

1. 登录 n8n Cloud。
2. 打开对应 workflow。
3. 修改节点配置。
4. 保存并 Publish。
5. 回到网页测试功能是否正常。
